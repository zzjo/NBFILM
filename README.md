## 关于 NbFilm

**NbFilm** 是一个创新的智能影视娱乐平台，将传统流媒体服务与 Web3 经济模型完美融合。在这里，你不仅是一位观众，更是生态系统的建设者——通过观看影片赚取代币奖励，解锁会员特权，参与平台生态治理。

> 💡 **核心理念**：让观影更有价值，让娱乐产生收益

---

## 核心功能

### 🎬 影视内容

| 功能 | 说明 |
|------|------|
| 海量资源 | 聚合多个优质影视源，涵盖电影、剧集、动漫等分类 |
| 智能搜索 | 支持关键词搜索，快速找到想看的内容 |
| 分类浏览 | 按类型、地区、年份等多维度筛选 |
| 播放体验 | 支持多播放源切换，适配 HLS 流媒体格式 |

### 👤 用户中心

- **Telegram 登录**：一键授权，快速进入
- **观看历史**：自动记录，追剧不错过
- **我的收藏**：一键收藏，随时观看
- **个人设置**：自定义偏好，打造专属体验

### 💰 代币经济系统 (NBT)

| 赚取方式 | 说明 |
|---------|------|
| 🎁 会员空投 | 开通会员即享每日代币空投 |
| ⏱️ 观影奖励 | 观看影片累积时长，自动获得代币奖励 |
| 🌟 会员加成 | 会员用户享受更高代币收益率 |
| 🔄 广告回购 | 平台每日 50% 广告收入用于回购销毁 NBT |

**代币用途**：
- 解锁付费影片
- 兑换会员权益
- 购买专属周边
- 提现至 BNB Chain 钱包

### 👑 会员体系

| 等级 | 权益 |
|------|------|
| 免费用户 | 基础观影、广告观看 |
| 会员用户 | 高收益加成、专属空投、抢先看等 |

### 🔐 安全功能

- **支付密码**：6 位数字密码，保护你的资产
- **钱包绑定**：支持绑定 MetaMask、Trust Wallet 等 BNB Chain 钱包
- **提现审核**：多重验证机制保障资金安全

**前端**
- Vue 3 + TypeScript
- Element Plus UI
- Video.js (HLS 播放器)

- Vite 构建

**后端**
- Go + Gin 框架
- GORM (ORM)
- Redis (缓存)
- MySQL (数据库)
- Gocolly (爬虫)
- Cron (定时任务)

**区块链**
- BNB Chain (BNB Smart Chain)
- NBT Token (BEP-20)

## 常见问题

### Q: 如何获得更多代币？
**A**: 
1. 开通会员，享受专属空投福利
2. 保持每日观影习惯，累积时长奖励
3. 邀请好友加入，双方获得奖励

### Q: 代币可以提现吗？
**A**: 可以。绑定外部 BNB Chain 钱包并设置支付密码后，可在钱包页面申请提现。

### Q: 观影奖励多久到账？
**A**: 系统会实时计算观影时长，代币奖励将在当日结算后自动发放到您的钱包。

### Q: 忘记支付密码怎么办？
**A**: 在钱包页面可以重新设置支付密码，需验证身份信息。
